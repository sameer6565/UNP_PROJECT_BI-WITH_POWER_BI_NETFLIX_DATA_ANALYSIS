# UNP_PROJECT_BI-WITH_POWER_BI_NETFLIX_DATA_ANALYSIS

1. Created a Netflix Userbase Analysis Dashboard using Power BI
2. Displays key metrics such as total users, total revenue, premium users, and ARPU
3. Analyzes users by subscription type (Basic, Standard, Premium)
4. Compares revenue contribution from each subscription plan
5. Segments users by device type (Laptop, Smartphone, Tablet, Smart TV)
6. Examines gender-wise, age-wise, and country-wise user distribution
7. Visualizes monthly revenue trends to track performance over time
8. Identifies business issues such as high discounts and low-profit segments
9. Provides actionable insights to improve pricing and profitability
10. Supports data-driven decision-making for management
